VMQemuVGA Enhanced Graphics Driver

This installer will install the VMQemuVGA kernel extension, which provides enhanced 3D graphics acceleration for QEMU virtual machines.

The installer will automatically:
• Detect your macOS version
• Install to the appropriate system location
